Usage

You can switch the style schemas. Choice will be saved in local storage or cookies.

Before, you need to create your alternatice stylesheets in '/sc' directory named st1.css, st2.css, st3.css etc. Then use this code to call switcher whith next arguments:

/* 
@ placeholder: DOM selector to append swicher menu 
@ path:		   path to you advanced style schemas
@ [5]:		   number of advanced stylesheets 
*/

$(function() {
var placeholder = $('#shell'),
	path = 'sc/';
 
style_switcher( placeholder, path, 5 );	
});

Of course, you can style it using css your like.
