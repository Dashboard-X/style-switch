About style switcher

  • Two way automatic savings in local storage or cookies if ls unavailible;
  • Worked on HTML 5 data attributes;
  • Simple usage;
  • Unlimited stylesheets;
  • Automated layout creation;
  • Multiple schemas mode;

Main

  • Supported browsers: IE6, IE7, IE8, IE9, IE10, FF, Opera, Chrome;
  • Requirements: jQuery 1.6.4 or higher, local storage pluggin;
  • version: 2.0 beta

Usage

You can switch the style schemas. Choice will be saved in local storage or cookies.

Before, you need to create your alternatice stylesheets in '/sc' directory named st1.css, st2.css, st3.css etc. Then use this code to call switcher whith next arguments:

/* 
 [Single mode]

@ placeholder: DOM selector to append swicher menu 
@ path:		   path to you advanced style schemas
@ [5]:		   number of advanced stylesheets 
*/

$(function() {
var placeholder = $('#shell'),
      path = 'sc/';
 
style_switcher( placeholder, path, 5 );	
});

/* 
 [Multiple mode]

@ placeholder: DOM selector to append swicher menu 
@ group:               unical name of schema group
@ path:		   unical path to you schemas folder contains styles
@ [5]:		   number of stylesheets you want to switch 
*/

$(function() {
var placeholder = $('#shell'),
      group = 'colors',
      path = 'sc/';
 
style_switcher( placeholder, path, 3 );	
});

Of course, you can style it using css your like.
